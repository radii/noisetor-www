tor.noisebridge.net

Noisebridge has joined forces with the Tor Project and the EFF to create a program where people can donate to a US-based non-profit organization and directly see their donations turn into Tor exit nodes.

How does it work?

  1. Donate money to Noisebridge using the button below
  2. Noisebridge sets up Tor nodes
  3. Together we make the Internet a more free place!

How do I find out more?
